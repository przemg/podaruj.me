Add shareable links to gift lists.

Every list gets a unique public link (short, friendly URL slug —
not a random ID). List owner can see and copy this link from
the list detail page with a "Share" button.

When someone opens the link:
- They see the list without logging in — name, description,
  occasion, event date with countdown, and all gifts
- The page looks beautiful and inviting, like receiving
  a gift list from a friend
- If they are not logged in, they are a "guest"
- No editing, no owner controls — just a clean read-only view
- Reserve buttons visible but not functional yet (next step)

The owner view and guest view should be different:
- Owner sees edit controls, settings, share button
- Guest sees a clean, read-only version focused on the gifts
