# Simplified Reservations - Initial Prompt

Simplify the reservation flow. Remove all email/OTP verification for guests.

## New flow when someone clicks Reserve:
- Enter nickname (required)
- If Buyer's Choice mode — toggle "Show my name to the list owner"
- Click confirm — done, reservation is instant

## Remove:
- OTP verification for guests
- Email input during reservation
- Any Resend leftovers
- Confirmation pages

## Keep:
- Privacy modes (Buyer's Choice, Visible, Full Surprise) — unchanged
- Cancel reservation
- Dashboard "My Reservations" for logged in users

Logged in users skip the nickname step — their display name is used.

Also update PROJECT.md to reflect this simpler flow.
